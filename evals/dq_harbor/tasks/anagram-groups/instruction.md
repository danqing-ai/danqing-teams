`/app/words.txt` has one word per line (lowercase ASCII).

Group anagrams: words that share the same sorted-letter signature.
Write `/app/groups.txt` where each line is a group: words sorted alphabetically, joined by commas.
Groups themselves sorted by the group's signature (sorted letters) ascending.
Skip empty lines. Singleton groups are included.

Example group line: `aet,ate,tea`

Do not ask the user any questions.
